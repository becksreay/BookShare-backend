# BookShare Backend
Backend service for a web-based platform for users to share, swap or give away books within their local community

## What it does
Provides API endpoints and data handling to support the frontend application.  

Handles book listing creation, storage, and retrieval, allowing users to add books and search for available listings.

## Tech stack
Django, Python, REST API

## Features
- API endpoints for creating and retrieving book listings  
- Database management for storing and retrieving listings
